    Algoritmo para sumar "n" numeros ingresados por teclado
  1. Inicio: Inicializar las variables suma=0 u numero
  2. Entrada: Solicita un número al usuario
  3. Ciclo: Iniciar el bucle mientras numero≠0
  4. Calculo: Si numero≠0 se le suma a la variable "suma"; suma+=numero
  5. Repeticion: El bucle vuelve al paso 2 y se vuelve a solicitar un numero
  6. Salida: Cuando el usuario ingresa el 0, se finaliza el bucle y se muestra el resultado de la variable "suma"
  7. Fin: Termina el algoritmo
